Jekyll helper

## Execute
```
bundle exec jekyll s
```

## Build
```
jekyll build
```

## Configure
```
$ENV:JEKYLL_ENV = "production"
$ENV:JEKYLL_ENV = $null
```

## Bundle clean
```
bundle clean --force
```
