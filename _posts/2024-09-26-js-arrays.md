## Find shared elements by property
```
const arr1 = [
  { id: 1, name: 'Alice' },
  { id: 2, name: 'Bob' },
  { id: 3, name: 'Charlie' }
];

const arr2 = [
  { id: 2, name: 'Bob' },
  { id: 4, name: 'David' }
];

// Find elements in arr1 that share the same `id` with arr2
const shared = arr1.filter(item1 =>
  arr2.some(item2 => item2.id === item1.id)
);
```
